# 💬 Full Stack Chat Application

A simple and clean **Full Stack Chat Application** built using Node.js and Express.
This project demonstrates API integration, user authentication (basic), and dynamic UI updates.

---

## 🚀 Features

* 🔐 User Login (Username & Password)
* 💬 Send and display messages
* 🧾 Chat history persists between users
* 👥 Messages show with username (group chat style)
* 🔄 Logout and re-login functionality
* 🎨 Clean and professional UI
* ⚡ Fast and lightweight (no database required)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Communication:** REST APIs

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/chat-app.git
cd chat-app
```

2. Install dependencies:

```
npm install express
```

3. Run the server:

```
node server.js
```

4. Open in browser:

```
http://localhost:3000
```

---

## 💡 How It Works

* Users log in with a username and password
* Messages are stored temporarily on the server (in-memory)
* When a new user logs in, previous messages are loaded
* Messages are displayed with usernames (group chat style)
* Users can log out and switch accounts easily
